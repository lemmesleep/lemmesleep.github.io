Window command line

2021-01-01
window

开启 window+R, 输入cmd

cmd：command line

基本操作:

cls: clear the screen

exit: exit the command line

cd: to a directory of the current directory.

cd..: to the previously directory.

cd../..: take me back 2 directory.

hit tab: autocomplete and alternate between the options.

dir: View the content of the current directory

dir path : View the content of the directory specified without moving to that directory.

dir /a: view the hidden file too 

                        `<DIR>`means that it is a directory.

mkdir: create a directory in the current directory

rmdir: delete the directory in the current directory if there is nothing inside of it.

  • What if there is thing inside it

    rmdir /s [directory name]: delete everything in the directory and the directory

    home and end: jump to the beginning and the end

PATH variable

when you list an absolute path you need to surround it with double quote.

Files/directory

attrib /? : attribute meanings

echo [text] > [.txt file] : if the text file is there, overwrite the content, if not create the file.

echo [text] >>[.txt file]: append the text to the .txt file

attrib -[property] +[property] : adding or deleting a property

copy and move files

copy a file: copy [source file] [folder to copy to]

transfer files of a folder to another folder: xcopy [source folder] [folder to copy to]

transfer everything(including subdirectories) to another folder: xcopy /s [source folder] [destination folder]

move [source] [destination]: move a directory to another

renaming a file

rename [directory] [new name]
