Database fundamentals

2020-12-27

这是一个希望暑期可以打工的少女在家里随便学的一点点东西。。。大概以后会尽量每周发个两三篇关于学习到的新知识啦什么的，偶尔也会更新一些自己的想法？那么就让我们开始吧！

Database

: is a collection if related information.
A lot of daily lists that we have such as shopping list, to-do list and Facebook’s user base are example of databases.
A database can be stored in different ways:

  1. on paper
  2. in a human’s head
  3. on a computer

Well, obviously a computer is much more efficient to store databases, mainly because of the multitude of database management system that can be be used to organize and keep track of a large amount of information

Database management system(DBMS)

: a software in which data of the databases are stored in the computer, it uses tables to organize these information.

There are 2 types of it:

  • Relational database management system(RDBMS) , which organize information into organized table, the relational databases uses a language called Structured Query Language(SQL), it is the language(commands) we make to interact with the RDBMS to perform the CRUD operation(create, retrieve, update, delete).

  • Non-relational database management system(NRDBMS)真是长啊2333, which does not use the SQL language, therefore called NoSQL databases, such as Mango DB and JSON(JavaScript Object Notation). Since there are no standards, each database has a different language to perform CRUD operations
