SQL fundamentals

The Structured Query Language is a language used for interacting with the Relational Database Management System.(RDBMS)

SQL is a hybrid language, it has 4 types of languages in one language, they are the DQL, the DDL, DCL and DML; they are used for query through, define the structure of the database, control the permission of a database and manipulate the data of the database.

  1. Data Query Language:

    • the DQL is used to obtain information that is already inside of the database.

  2. Data Definition Language:

    • the DDL is used to define the schema of the database, for example the name of the tables in the databases and the columns that each table has.

  3. Data Control Language

    • the DCL is used to set restriction on the data, to control access to the data from the database.—permission management

  4. Data Manipulation Language

    • the DML is used to manipulate the data in the database, namely to insert, update and delete an entry of the database.

Queries

A querie is a set of instructions given to the Database management system to execute, a set of actions or a bloc of code…to retieve data that you need from the database.

  • The database management system will then return to you a set of data according to the code that you have typed.
